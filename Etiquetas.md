# Etiquetas

## Tipo de tarea

- **bug**: ![#d73a4a](https://placehold.co/15x15/d73a4a/d73a4a.png) - Un fallo en el código que necesita ser solucionado.
- **enhancement**: ![#a2eeef](https://placehold.co/15x15/a2eeef/a2eeef.png) - Mejora de una característica existente del proyecto.
- **documentation**: ![#0075ca](https://placehold.co/15x15/0075ca/0075ca.png) - Referente a la mejora o creación de documentación del proyecto.
- **help wanted**: ![#ffcc00](https://placehold.co/15x15/ffcc00/ffcc00.png) - Se necesita ayuda con una tarea o problema específico.
- **good first issue**: ![#008672](https://placehold.co/15x15/008672/008672.png) - Problemas que son adecuados para nuevos contribuyentes.

## Prioridad

- **priority: high**: ![#cb2431](https://placehold.co/15x15/cb2431/cb2431.png) - Debe ser abordado de inmediato.
- **priority: medium**: ![#ffcc00](https://placehold.co/15x15/ffcc00/ffcc00.png) - Importante, pero no urgente.
- **priority: low**: ![#57606a](https://placehold.co/15x15/57606a/57606a.png) - Puede ser abordado en un futuro.

## Estado

- **in progress**: ![#ffa500](https://placehold.co/15x15/ffa500/ffa500.png) - Tareas que se están desarrollando.
- **to do**: ![#f6f8fa](https://placehold.co/15x15/f6f8fa/f6f8fa.png) - Tareas que aún no se han comenzado.
- **done**: ![#28a745](https://placehold.co/15x15/28a745/28a745.png) - Tareas que se han completado.
